# Portafolio Web

Este es el repositorio base para el sitio web del portafolio de nuestro equipo de desarrollo.

## 📁 Estructura

```
portafolio-web/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── assets/
│   └── img/
└── README.md
```

## 🛠️ Personalización

Agrega tu información en `index.html` en las secciones correspondientes:
- Sobre nosotros
- Proyectos
- Contacto

Estiliza con `css/styles.css` y agrega interactividad con `js/script.js`.
